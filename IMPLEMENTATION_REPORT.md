## Features Delivered

### 1. Document & PDF Tools
- **Document Scanner**: Scan physical documents using CameraX + ML Kit.
- **PDF Compressor**: Reduce PDF file size.
- **PDF Password Remover**: Decrypt PDF files.

### 2. Image Tools
- **Image Compressor**: Compress/Resize images with quality slider.

### 3. Calculators
- **SIP Calculator**: Calculate Mutual Fund returns.
- **Simple Interest**: Basic interest calculation.
- **Compound Interest**: Compound growth calculation.

### 4. Utility Tools
- **World Clock**: Hierarchical Region/Country/City timezone picker.
- **ASCII Table**: Reference grid for developer codes.
- **Message Analyzer**: Scans SMS inbox for OTPs and Junk.

### 5. Dictionary
- **Hindi-English**: Offline dictionary with Synonyms/Antonyms.

### 6. Settings
- **Preferences**: Theme (Dark/Light), Default Quality.
- **Maintenance**: Clear Cache functionality.

### 7. Core & UI
- **Design System**: Glassmorphism, Dark/Teal/Purple Theme.
- **Navigation**: Modular functionality.
- **App Icon**: Custom Vector Drawable (Neon Cross on Dark Shield).

## Technical Details
- **Architecture**: MVVM + Clean Architecture + Multi-Module.
- **Tech Stack**: Kotlin, Jetpack Compose, Hilt, ML Kit, CameraX, PDFBox.
- **Min SDK**: 36 (Android 16+ as requested).

## Next Steps for User
1. Run the app on an Android 16 Emulator or Device.
